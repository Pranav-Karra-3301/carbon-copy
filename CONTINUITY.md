# Carbon Copy - Continuity Ledger

## Goal
Build "Carbon Copy" - a vintage 1970s form UI component library for React inspired by the Chevrolet dealer order forms.
- Success criteria: Installable via `npm install @pranav-karra/carbon-copy`
- Demo page showcasing all components

## Constraints/Assumptions
- React + TypeScript
- CSS Modules for styling
- Next.js for demo site
- tsup for library bundling

## Key Decisions
- Library name: `@pranav-karra/carbon-copy`
- Styling: CSS Modules (authentic vintage feel)
- Landing Page: Showcase demo as the main page
- 7 core components built
